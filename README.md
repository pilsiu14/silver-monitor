# Silver Squeeze Monitor

Personal dashboard tracking COMEX silver inventory, Shanghai premium, and short squeeze indicators.

## Tech stack

- Next.js 14 (App Router)
- Supabase (PostgreSQL)
- Tailwind CSS
- Recharts
- Cloudflare Pages / Vercel

## Data sources

- CME Group Daily Stocks Report (XLS scraping)
- gold-api.com (spot price)
- Shanghai Gold Exchange (HTML scraping)
- CFTC Commitments of Traders
